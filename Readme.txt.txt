Technology - 
1) HTML
2) Tailwind CSS
3) JavaScript
4) Database - LocalStorage

Key Features:
1) Product Listing:
Display a list of products available for purchase.
2) Add to Cart:
Enable users to add products to a shopping cart.
3) User Authentication:
Allow users to log in and sign up.
4) User Dashboard:
Display the user's name on the dashboard after logging in.
